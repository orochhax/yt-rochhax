## 🎯 Padrões de TÍTULO
| Padrão | Exemplo | Por que funciona |
|---|---|---|
| Número específico e "quebrado" | `R$31.096,42` | parece real, não inventado |
| Número gigante | `US$40 BILHÕES` | escala = curiosidade |
| Esforço baixo / atalho | `mais PREGUIÇOSO`, `comece do zero` | derruba a barreira de entrada |
| Convite a copiar | `Apenas ME COPIE` | promete resultado replicável |
| Prova em 1ª pessoa | `USEI`, `EU não posso viver` | credibilidade / autenticidade |
| Pergunta de curiosidade | `Por que o Google...` | abre um loop na cabeça |
| Reticências de hype | `...Inteligente demais...` | frase incompleta puxa o clique |
| Sub-hook entre parênteses | `(comece do zero)` | segunda promessa no mesmo título |
| Alerta / aversão à perda | `Não fique na mão da...` | medo de ficar pra trás |
| CAPS seletivo | palavras-chave em maiúscula | guia o olho para o que importa |
